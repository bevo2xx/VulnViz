Project Overview

VunViz is a student prototype project designed to automatically scan websites for vulnerabilities, link findings to known CVEs, and generate clear reports including proof-of-concept scripts and manual testing guidance. The project is intended for learning and demonstration purposes.

Goals

  • Provide a web interface to configure and run vulnerability scans.
  • Automate basic scanning using open-source tools (e.g., Nmap, ZAP, Nikto, SQLMap).
  • Map discovered vulnerabilities to CVEs and provide severity scoring.
  • Generate PDF/HTML reports including POC scripts and manual testing guidance.
  • Offer a student-friendly learning mode with explanations for each vulnerability.

Key Features

  • Web UI for scan configuration and result dashboard.
  • Automatic CVE mapping and severity scoring.
  • Auto-generated proof-of-concept (POC) scripts.
  • Manual test guidance for safe testing.
  • Report generation in PDF and HTML formats.
  • Learning mode explanations for students.

Architecture

Web UI <---> Backend API <---> Scan Engines <---> CVE Enrichment <---> Reports

Web UI: Configure scans, view dashboard, download reports.
Backend API: Orchestrates scans, parses results, enriches findings.
Scan Engines: Nmap, ZAP, Nikto, SQLMap (open-source tools).
CVE Enrichment: Map findings to known vulnerabilities with version checks.
Reports: Generate downloadable PDF/HTML with POCs and guidance.

How the Prototype Works

  1. User configures target URL and scan type in the web interface.
  2. Backend runs selected scan engines.
  3. Findings are parsed and normalized.
  4. Findings are enriched with CVEs, severity scoring, POC scripts, and learning guidance.
  5. Results are displayed on the dashboard and can be exported as PDF or HTML reports.

Timeline & Costs (Student Prototype)

  • Development time: ~120 hours (4-6 weeks).
  • Internet/data: ~500 EGP (medium usage for downloads & scans).
  • Hosting / cloud VM: ~1,000 EGP (optional, prototype can run locally).
  • Miscellaneous (domain, backups, small utilities): ~500 EGP.
  • Total estimated cost: ~8,500 EGP (including student effort).

Student-Friendly Notes

  • The prototype only runs safe, authorized scans.
  • All data is stored locally; sensitive info is not shared.
  • Learning mode provides explanations for understanding vulnerabilities.
  • Reports include visual summaries and risk guidance.

Conclusion

VunViz demonstrates how automated scanning, CVE enrichment, and reporting can be combined in a simple student project. It emphasizes safe practices, learning, and clear reporting.
